== Description ==

Simple Slider plugin built on Nivo Slider with default themes.
Simply use the shortcode [vds-shortcode]

== Installation ==


1. Upload the vdSlider.zip file - or Use the plugin install in admin panel to locate "vdSlider"
2. Install and Activate the plugin through the 'Plugins' menu in WordPress
3. Add slideshows through the new menu in the admin panel called "vdSlider"
4. Go to the page you wish to show your slideshow and add the shortcode [vds-shortcode]
